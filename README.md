# '웹 접근성'
장애인, 고령자 등이 웹 사이트에서 제공하는 정보에 비장애인과 동등하게 접근하고 이해할 수 있도록 보장하는 것

---

## 📍논리적인 순서
* 프레임, 콘텐츠는 논리적인 순서로 제공해야 한다.
* 화면에 보여지는 콘텐츠의 순서가 아닌 **검색  봇이나 웹 브라우저가 콘텐츠를 탐색하고 이해할 수 있는 선형화된 순서**
###### ex) 모바일 메뉴 제작 시, 내용 탐색 후 마지막에 '닫기 버튼'에 도달하도록 마크업
